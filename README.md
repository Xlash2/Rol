# Rol
An arcade-style web game for mobile using gyro controls. Installable as a PWA. Play [here](https://www.google.com).

<img width="360" height="640" alt="Screenshot_20260426_123959_Chrome" src="https://github.com/user-attachments/assets/8b1af0a8-b6a6-43e1-a548-20ba61d3457d" />

## Setup

```
npm install
npx vite --host
```
Then access on your favorite mobile device.

## Libraries Used

- [Babylon.js](https://www.babylonjs.com/): Graphics rendering and physics (provided by the built-in Havok plugin)
- [Svelte](https://svelte.dev/): UI rendering and reactivity
- [Bulma](https://bulma.io/): UI
- [howler.js](https://howlerjs.com/): Audio
- [Vite](https://vite.dev/): Building and development environment
- [vConsole](https://github.com/tencent/vconsole): Debugging
